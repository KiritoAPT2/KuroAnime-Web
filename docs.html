Aplicación Android para ver anime subtitulado y latino mediante un sistema de extensiones — creada por KiritoAPT2

Introducción

KuroAnime es una aplicación Android enfocada en el consumo de anime subtitulado y doblado al español latino mediante un sistema moderno de extensiones. El proyecto nace como una alternativa liviana y sin anuncios para la comunidad hispanohablante.

Su objetivo principal es ofrecer una experiencia limpia y rápida para acceder a contenido desde múltiples fuentes dentro de una sola aplicación, con diseño OLED para ahorrar batería en pantallas AMOLED.

Filosofía del Proyecto

KuroAnime fue diseñado pensando en la simplicidad y la libertad del usuario. En lugar de depender de una única fuente de contenido, utiliza un sistema modular basado en extensiones JavaScript y Kotlin que permite agregar nuevas plataformas fácilmente.

La aplicación es completamente gratuita, sin anuncios ni rastreo. Todo el contenido se obtiene directamente de fuentes públicas mediante scraping, sin intermediarios ni servidores propios.

Compatibilidad

KuroAnime está desarrollada para dispositivos Android con versión mínima API 24 (Android 7.0) y apunta a API 35 (Android 15).

Para instalar la aplicación:

# Compilar desde código fuente
git clone https://github.com/KiritoAPT2/KuroAnime.git
cd KuroAnime
./gradlew assembleDebug

# APK generado en:
# app/build/outputs/apk/debug/app-debug.apk

También puedes descargar la última versión precompilada desde GitHub Releases.

Sistema de Extensiones

Uno de los pilares principales de KuroAnime es su arquitectura basada en extensiones. Cada extensión es responsable de obtener información desde sitios compatibles y transformarla en contenido reproducible dentro de la aplicación.

Actualmente hay dos tipos de extensiones:

  • JavaScript (QuickJS) — cargadas desde assets/extensions/, ejecutadas con un motor JS embebido. Prioridad sobre Kotlin.
  • Kotlin nativas — implementadas directamente en el código usando Jsoup o Ktor para sitios que requieren lógica más compleja.

El sistema permite:

  • Buscar contenido por palabras clave
  • Obtener los últimos episodios agregados
  • Navegar por categorías y géneros
  • Cargar información detallada de cada anime
  • Generar enlaces de reproducción desde múltiples servidores

Funciones de las Extensiones

Cada extensión implementa funciones específicas para integrarse correctamente con la aplicación:

FunciónDescripción
getLatest(page)Obtiene los animes más recientes
search(query, page)Busca animes por nombre
getAnimeInfo(url)Obtiene detalles completos de un anime
getEpisodes(url)Lista todos los episodios disponibles
getVideoSources(url)Genera enlaces de reproducción

Fuentes Soportadas

KuroAnime incluye soporte para múltiples fuentes de contenido, cada una implementada como una extensión independiente:

  • AnimeFLV — scraping vía Jsoup desde animeflv.net
  • Latanime — scraping desde latanime.org para doblaje latino
  • TioAnime — API JSON desde jimov-api.vercel.app
  • Aniyae — scraping vía Ktor desde open.aniyae.net

Las extensiones se agregan automáticamente al iniciar la aplicación. No requieren configuración manual.

Estructura del Proyecto

El repositorio está organizado para separar correctamente el sistema de extensiones, la UI y la configuración:

KuroAnime/
├── app/
│   ├── build.gradle.kts
│   └── src/main/
│       ├── assets/extensions/   — .js para QuickJS
│       ├── java/com/kuroanime/
│       │   ├── data/            — modelos, HTTP client
│       │   ├── extension/       — motor de extensiones
│       │   ├── player/          — ExoPlayer + resolvers
│       │   └── ui/              — Compose screens
│       └── res/                 — iconos, temas
├── gradle/
│   └── libs.versions.toml      — dependencias
├── build.gradle.kts
└── settings.gradle.kts

Futuro del Proyecto

El objetivo a largo plazo de KuroAnime es convertirse en una de las principales aplicaciones open source para ver anime en Android. El proyecto busca expandir el ecosistema de extensiones, mejorar la interfaz, optimizar el rendimiento y agregar nuevas funcionalidades como:

  • Soporte para download de episodios
  • Lista de seguimiento personalizada
  • Notificaciones de nuevos episodios
  • Más extensiones de la comunidad
  • Mejoras en el reproductor de video
